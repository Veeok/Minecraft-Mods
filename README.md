# Minecraft Mods

Small Minecraft mods by Veok.

## Mods

- `AutoToolSwitcher` - swaps to the right hotbar tool, protects low-durability gear, supports slot filters, combat helpers, profiles, and a Mod Menu config screen.
- `ServerRestartCommand` - adds server-side `/restart` and `/restartserver` commands with a cross-platform relauncher for dedicated servers.

More tiny quality-of-life mods can live here later.
